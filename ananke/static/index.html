Ananke is a Linux container for experimenting with data science tools using the Jupyter notebook. Notebooks can be run stand-alone on a single machine, or as an ad-hoc cluster of machines on the same local area network using Apache Spark. Python libraries included are scikit-learn, NLTK, NumPy, matplotlib, Pandas, Gensim, pyLDAvis...

(Ananke is a moon of Jupiter, and also the Greek godess of constraint and necessity.)
